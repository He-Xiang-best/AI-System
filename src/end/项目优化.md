# 证件识别

待完善的功能内容有：

- 身份证头像识别
- 中文字符训练
- 姓名、民族、性别、出生日期等定位识别

其次，项目识别证件号的算法模型识别率受证件图片质量影响，图片的证件号如果数字0是带有斜杠的会影响识别结果，且上传的证件图片环境不满足纯色背景等要求，将会识别失败。

项目中待优化部分：将上传的证件照图片给予剪裁功能，提高识别度
